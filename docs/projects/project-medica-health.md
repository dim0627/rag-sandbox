## 背景・課題

メディカヘルス様は3病院でそれぞれ異なる電子カルテを運用しています。

1. **データ連携**: 患者が別病院を受診する際、過去の診療情報が参照できない
2. **重複検査**: 連携不足により、同じ検査を重複して実施するケースがある
3. **地域医療連携**: 地域の診療所との情報連携ができていない
4. **統計・分析**: グループ全体の診療データを横断的に分析できない

## 提案内容

### 提案システム構成

1. **医療情報連携基盤**: MedConnect（当社パッケージ）
2. **データ標準化**: HL7 FHIR対応
3. **インフラ**: オンプレミス（院内DC）+ AWS（DR用）
4. **セキュリティ**: 3省2ガイドライン準拠

### 導入範囲

- 電子カルテ連携（SS-MIX2標準化ストレージ）
- 患者ID統合管理
- 診療情報ビューア
- 地域医療連携機能
- 統計・分析ダッシュボード

### 見積金額

### 期待効果

- 重複検査の削減: 年間約800万円のコスト削減
- 診療の質向上: 過去の診療情報を即座に参照可能
- 地域連携強化: 紹介・逆紹介の効率化
