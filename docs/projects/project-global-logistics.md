## 背景・課題

グローバル物流様は関東圏に5つの物流センターを運営しています。現在の課題:

1. **在庫精度**: 棚卸差異率が1.2%と業界平均（0.5%）より高い
2. **作業効率**: ピッキング作業が紙ベースで、1件あたり平均8分かかっている
3. **拠点間連携**: 各拠点で異なるExcel管理をしており、在庫の横持ち判断に時間がかかる
4. **EC対応**: 新規にEC物流の受託を計画しているが、現システムでは対応困難
