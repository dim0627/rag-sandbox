# 販売管理システム（SalesForce Pro）操作マニュアル

## 概要

SalesForce Proは当社の販売活動を一元管理するシステムです。案件管理、見積作成、受注処理、売上管理などの機能を提供します。

## ログイン方法

1. ブラウザで https://sales.example.co.jp にアクセス
2. 社員番号とパスワードを入力
3. 多要素認証コードを入力

※初回ログイン時はパスワード変更が必要です。

## 案件登録

### 新規案件の登録手順

1. メニューから「案件管理」→「新規登録」を選択
2. 以下の必須項目を入力:
   - 案件名: 顧客名＋案件概要（例: ABC商事様_基幹システム刷新）
   - 顧客名: 顧客マスタから選択
   - 案件ステータス: 引合/提案中/見積提出/内示/受注/失注から選択
   - 想定売上金額: 税抜金額を入力
   - 受注予定日: 見込み日を入力
   - 担当者: 自動で入力済み（変更可能）
3. 「保存」ボタンをクリック

### 案件ステータスの定義

| ステータス | 定義 | 確度 |
|-----------|------|------|
| 引合 | 顧客から問い合わせを受けた段階 | 10% |
| 提案中 | 提案書を作成・提出している段階 | 30% |
| 見積提出 | 見積書を提出した段階 | 50% |
| 内示 | 口頭で受注の内示を受けた段階 | 80% |
| 受注 | 正式に契約を締結した段階 | 100% |
| 失注 | 他社に決定または案件消滅 | 0% |

## 見積作成

### 見積書の作成手順

1. 案件詳細画面から「見積作成」ボタンをクリック
2. 見積番号は自動採番されます（形式: QT-YYYYMM-XXXX）
3. 明細を入力:
   - 品目コードを入力すると品名・単価が自動入力
   - カスタム品目は「その他」を選択して手入力
4. 値引きがある場合は「値引行追加」から入力
5. 「プレビュー」で確認後、「PDF出力」

### 見積有効期限

- 標準: 見積日から30日
- 大型案件（1,000万円以上）: 見積日から14日
- 有効期限は手動で変更可能

## 受注処理

### 受注登録の手順

1. 案件詳細画面でステータスを「受注」に変更
2. 以下の情報を入力:
   - 受注日
   - 受注金額（税抜）
   - 契約書番号
   - 納品予定日
3. 「受注登録」ボタンをクリック
4. 経理部門に自動通知されます

### 注意事項

- 受注金額が見積金額と10%以上乖離する場合、上長承認が必要です
- 1,000万円以上の案件は部長承認が必要です
- 5,000万円以上の案件は役員承認が必要です

## レポート・分析

### 標準レポート

- 月次売上レポート: 毎月1日に自動生成
- 案件パイプラインレポート: リアルタイム更新
- 営業担当別実績レポート: 週次更新

### カスタムレポートの作成

1. 「レポート」→「新規作成」を選択
2. テンプレートを選択（案件/売上/顧客）
3. 表示項目と条件を設定
4. 「保存」でマイレポートに追加

## よくあるトラブル

### Q: 案件が検索で見つかりません

A: 検索条件を確認してください。デフォルトでは「担当者=自分」になっています。「すべての案件」に切り替えるか、担当者条件を外してください。

### Q: 見積のPDF出力でエラーになります

A: ポップアップブロックが原因の可能性があります。ブラウザの設定で sales.example.co.jp のポップアップを許可してください。

### Q: 過去のデータを修正したい

A: 当月分は修正可能です。前月以前のデータ修正は、経理部門への依頼が必要です。修正依頼フォームから申請してください。
